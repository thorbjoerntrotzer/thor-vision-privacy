# THOR Vision – Privacy Policy

THOR Vision is a private personal home security application.

## Data Collection
This app does not collect, share, or transmit any personal data to third parties.

## Data Storage
All data is stored exclusively on the user's local Raspberry Pi within their private home network. No data leaves the local network.

## Purpose
The app is used solely for private monitoring of the user's own property and vehicles.
