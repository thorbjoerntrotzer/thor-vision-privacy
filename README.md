# THOR Vision – Privacy Policy

THOR Vision is a private home security app for monitoring 
vehicle access on personal property.

## What the app displays
The app displays vehicle-related data (such as license plates, 
associated labels, and images) provided by a locally hosted system 
within a private home network.

## Data Processing
The app does not collect or transmit data to external servers or 
third parties.

The app itself does not independently generate or collect personal data. 
All data is processed locally within the user's private network 
on a Raspberry Pi system.

## Push Notifications
The app uses Apple Push Notification services (APNs). A device token is 
stored on the user's local server solely for the purpose of delivering 
notifications. This token is not shared with third parties.
